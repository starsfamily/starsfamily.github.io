linuxMint下安装markdown编辑器：typora

Feb 29, 2020點擊

orwget -qO - https://typora.io/linux/public-key.asc | sudo apt-key add -

echo -e "\ndeb https://typora.io/linux ./" | sudo tee -a /etc/apt/sources.list
sudo apt-get update

在使用apt-get 下载typora的过程中，下载速度极慢，这时候就需要apt-fast这个软件了。
apt-fast 通过使用多线程下载来给apt-get 提速。下载效果出奇的好。
