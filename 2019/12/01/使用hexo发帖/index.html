使用hexo发帖

Dec 01, 2019博客网络點擊

在安装hexo前需要先装好git和npm两个软件，如果不确定是否已经装好，可以先在终端使用git version查看。

如果没有直接使用命令sudo apt-fast install git 或者sudo apt-fast install npm即可。

安装hexo的命令为npm install -g hexo-cli或npm install hexo。

更改镜像在安装hexo，下载的速度更快些：

npm config set registry https://registry.npm.taobao.org//更换成淘宝npm镜像
// 配置后可通过下面方式来验证是否成功
npm config get registry
// 或
npm info express

安装以后，可以使用以下两种方式执行 Hexo：

  1. npx hexo//会出现npx的缓存，需等待。

  2. 使用whereis hexo 找出安装的路径

    将 Hexo 所在的目录下的 node_modules（有可能不是该文件夹） 添加到环境变量之中即可直接使用 hexo：

需要在项目根目录下执行命令行
npm i npm -g 更新npm的版本
npm install hexo-deployer-git -save 安装hexo部署到github的插件

sudo hexo new “xxx”//生成文档及相应的图片文件夹

需添加 \etc\hosts文件的dns解析地址。github.com和git.coding.net的dns解析IP
